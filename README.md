# Terraform学習サイト

Terraform の基礎から実践までを学べるインタラクティブな学習サイトです。

## セットアップ

```bash
npm install
npm run dev
```

## コンテンツ

- 基本概念（Provider / Resource / Module 等）
- ワークフロー・コマンド・プロバイダー
- コンテナ構築（ECS / EKS / GKE / Cloud Run）
- ECS アーキテクチャ解説
- VPC 構築ステップバイステップガイド
- VPC & コンテナ実践
- AWS 連携（ECR / S3 / DynamoDB）
- Cognito 認証
- Zscaler ZPA App Connector
- サーバーレス・マルチリージョン・デプロイ実践
- Git / GitHub 連携・ベストプラクティス
- 演習問題 20 問
